"HEX"
(Пока что не) игра в мире шестиугольных призм.


СИСТЕМНЫЕ ТРЕБОВАНИЯ============================================================

* Компьютер с операционной системой Windows XP или выше.
* Поддержка OpenGL 3.3 в операционной системе/драйверах.
* Видеокарта с 512 Mb оперативной памяти.
* 1 Gb оперативной памяти.
* Экран с разрешением не меньше чем 800x600.
* Наличие не менее одной руки, не менее одного глаза, не менее одной головы у
  пользователя.


УПРАВЛЕНИЕ======================================================================

WASD - перемещаться.
Z - включить/выключить режим полёта.
C - лететь вниз.
SPACE - лететь вверх/прыгать.
E - открыть меню выбора блока.
MOUSE1 - ломать.
MOUSE2 - строить.


НАСТРОЙКИ=======================================================================

Игра читает файл настроек config.json. Можно руками его отредактировать, если
есть понимание что есть что.


СОХРАНЕНИЕ======================================================================

Игра сохраняет мир в каталог "world". Если в мире что-то не устраивает, можно
удалить файлы из этого каталога или отредактировать их.
